# Movies Website (React.js)

A React-based movies website that displays the latest and popular movies using **The Movie Database (TMDB) API**.  
Users can browse movies and add or remove their favorite movies.

---

## Features
- View latest & popular movies
- Add movies to favorites
- Remove movies from favorites
- Responsive UI
- Data fetched using TMDB API

---

## Tech Stack
- React.js
- JavaScript (ES6)
- TMDB API
